SQL Data Types

SQL supports a wide range of data types for defining columns in tables. These data types can be categorized into several groups: numeric, string, date and time, binary, and other types. Here is a comprehensive list of common SQL data types across various SQL databases:

Numeric Data Types

Integer Types

  • INT or INTEGER: A standard integer.
  • SMALLINT: A smaller range integer.
  • TINYINT: A very small integer (supported in MySQL).
  • BIGINT: A large range integer.

Decimal Types

  • DECIMAL(p, s) or NUMERIC(p, s): Fixed-point numbers where p is precision (total number of digits) and s is scale (number of digits to the right of the decimal point).
  • FLOAT: Floating-point number with single precision.
  • DOUBLE or DOUBLE PRECISION: Floating-point number with double precision.
  • REAL: Single precision floating-point number (commonly used in PostgreSQL).

Bit Types

  • BIT: A bit field or binary field.

String Data Types

Character Types

  • CHAR(n) or CHARACTER(n): Fixed-length string where n is the number of characters.
  • VARCHAR(n) or CHARACTER VARYING(n): Variable-length string with a maximum length of n characters.
  • TEXT: Variable-length string with no specific upper limit (size may vary based on the database system).

Binary Types

  • BINARY(n): Fixed-length binary data.
  • VARBINARY(n): Variable-length binary data with a maximum length of n bytes.
  • BLOB: Binary Large Object for storing large binary data.

Date and Time Data Types

Date and Time

  • DATE: Stores a date (year, month, day).
  • TIME: Stores a time of day (hours, minutes, seconds).
  • DATETIME: Stores both date and time.
  • TIMESTAMP: Stores both date and time, often with timezone information.
  • YEAR: Stores a year value (supported in MySQL).

Interval Types (PostgreSQL specific)

  • INTERVAL: Represents a time span (years, months, days, hours, minutes, seconds).

Other Data Types

Boolean

  • BOOLEAN: Stores TRUE or FALSE.

UUID

  • UUID: Universal Unique Identifier (supported in PostgreSQL).

Enumerated Types

  • ENUM('value1', 'value2', ...): A string object that can only have one value, chosen from a list of allowed values (supported in MySQL).

JSON Types

  • JSON: Stores JSON (JavaScript Object Notation) data.
  • JSONB: Binary JSON storage format for JSON data (PostgreSQL).

Geometric Types (PostgreSQL specific)

  • POINT: Represents a geometric point.
  • LINE: Represents a geometric line.
  • LSEG: Represents a line segment.
  • BOX: Represents a rectangular box.
  • PATH: Represents a geometric path.
  • POLYGON: Represents a polygon.
  • CIRCLE: Represents a circle.

XML Types

  • XML: Stores XML data.

Arrays (PostgreSQL specific)

  • ARRAY: Stores an array of values.
