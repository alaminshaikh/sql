Functions for Operations

i) Display the day from the date of India's independence

To display the day like "Monday", "Tuesday", etc., from the date when India got independence (August 15, 1947), you can use the DAYNAME() function in SQL.

Example:

SELECT DAYNAME('1947-08-15');

ii) Display specified number of characters from a string

To display a specified number of characters from a particular position of the given string, you can use the SUBSTRING() function in SQL.

Example:

SELECT SUBSTRING('Hello World', 1, 5);

iii) Display the name of the month of your birth

To display the name of the month in which you were born, you can use the MONTHNAME() function in SQL.

Example:

SELECT MONTHNAME('1990-05-15');

iv) Display your name in capital letters

To display your name in capital letters, you can use the UPPER() function in SQL.

Example:

SELECT UPPER('your_name');
