SQL ALTER TABLE Examples

SQL Command Explanation
ALTER TABLE users
ADD email_verified BOOLEAN DEFAULT false;		 Adds a new column email_verified with a data type BOOLEAN and a default value of false.
ALTER TABLE users
ADD email VARCHAR(255) UNIQUE;		 Adds a new column email with a data type VARCHAR(255) and a unique constraint.
ALTER TABLE orders
ADD customer_id INT,
ADD CONSTRAINT fk_customer
FOREIGN KEY (customer_id)
REFERENCES customers(id);		 Adds a new column customer_id to the orders table and creates a foreign key constraint named fk_customer, which references the id column in the customers table.
ALTER TABLE products
ADD INDEX idx_product_name (name);		 Adds an index named idx_product_name to the name column in the products table.
ALTER TABLE employees
ADD id INT AUTO_INCREMENT PRIMARY KEY;		 Adds a new column id to the employees table, sets it as an auto-incrementing primary key column.
